id: accessibility
label: "Accessibility (a11y)"
---

Review UI code (HTML, JSX, templates, components) for accessibility issues. Ensure: semantic HTML elements are used instead of generic divs/spans, images have meaningful alt text, form inputs have associated labels, ARIA attributes are used correctly, keyboard navigation works, colour contrast meets WCAG AA standards, and focus management is correct. If the project has no UI code, report that and skip.
